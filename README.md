# RaptorGate - Firewall Nowej Generacji

**RaptorGate** to kompletny system zaawansowanego firewalla nowej generacji (NGFW), łączący wielowarstwową inspekcję ruchu sieciowego na warstwach L2–L7 modelu OSI z panelem administracyjnym dostępnym przez przeglądarkę. System integruje klasyfikację aplikacji niezależną od numerów portów, system zapobiegania włamaniom (IPS), detektowanie anomalii opartej na uczeniu maszynowym oraz reaktywny dashboard do zarządzania regułami i alertami w czasie rzeczywistym.
